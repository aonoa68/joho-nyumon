## 何の話か

音声をその場で聞き取り、別の言語へほぼ遅延なく翻訳して読み上げる——
こうした「リアルタイム翻訳AI」が、実用レベルに近づいています。

これまで「翻訳」は、

```text
話す → 録音 → 文字起こし → 翻訳 → 読み上げ
```

と段階を踏むものでした。各段階に時間がかかり、会話のテンポでは使えませんでした。

リアルタイム翻訳AIは、この流れをひとつながりに、会話と同じ速度で処理します。
言語の壁が「待ち時間ゼロ」で越えられる、という点が新しさです。
