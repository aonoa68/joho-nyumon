# 情報入門

北星学園大学「情報入門」（2026年度）の授業資料ページです。

このページでは、各回の授業資料・追加資料・課題をまとめています。
資料はいつでも見返せます。欠席した回もここで確認してください。

!!! note "この授業で大切にしていること"
    この授業のゴールは「Word や Excel の操作を覚えること」ではありません。
    操作は、必要になったら調べたり、AI に頼んだりすればできます。

    大切なのは、**「何をしたいか」を考える力**と、**情報を正しく読み・扱う力**です。
    道具に使われるのではなく、道具を使う人になりましょう。
